# Local Dia Workflow

This fork keeps the Dia-loaded extension in `local-extension/`.

The main Dia profile is registered to load this unpacked extension from
`/Users/eli/Dev/minimal-twitter-fork/local-extension`.

Use `scripts/start-dia` to open the normal `/Applications/Dia.app`.
Use `scripts/restart-dia` after changing extension files so Dia reloads the fork.
